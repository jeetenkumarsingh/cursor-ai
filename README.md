# Second Home Stories - WordPress Theme

A modern, responsive WordPress theme inspired by Second Home Stories. This theme features a clean design, mobile-first approach, and customizable hero section.

## Features

- **Responsive Design**: Mobile-first approach with breakpoints at 768px and 480px
- **Modern Layout**: Card-based design with hover effects and smooth transitions
- **Customizable Hero Section**: Easy to customize through WordPress Customizer
- **SEO Optimized**: Proper HTML5 structure and semantic markup
- **Widget Ready**: Sidebar and footer widget areas
- **Translation Ready**: Internationalization support
- **Customizer Integration**: Live preview of theme customizations
- **Clean Typography**: Uses Inter font family for modern readability

## Installation

1. Download the theme files
2. Upload the theme folder to `/wp-content/themes/` directory
3. Activate the theme through the 'Appearance > Themes' menu in WordPress
4. Customize the theme through 'Appearance > Customize'

## Required Plugins

- **WordPress 5.0+** (for full compatibility)
- **Classic Editor** (optional, for better content editing experience)

## Theme Customization

### Hero Section
- **Hero Title**: Customize the main heading text
- **Hero Subtitle**: Customize the subtitle text
- **CTA Button Text**: Customize the call-to-action button text
- **CTA Button Link**: Set the destination URL for the button

### Colors
- **Primary Color**: Customize the theme's primary color scheme

### Navigation
- **Primary Menu**: Set up navigation through Appearance > Menus
- **Footer Menu**: Optional footer navigation

### Widgets
- **Sidebar**: Add widgets to the main sidebar
- **Footer Widget Area**: Add widgets to the footer

## Template Files

- `index.php` - Main template for blog posts
- `single.php` - Individual post template
- `page.php` - Static page template
- `archive.php` - Category, tag, and date archive templates
- `search.php` - Search results template
- `404.php` - Error page template
- `comments.php` - Comments display and form
- `header.php` - Site header
- `footer.php` - Site footer
- `sidebar.php` - Sidebar template
- `searchform.php` - Search form template
- `functions.php` - Theme functions and setup
- `style.css` - Main stylesheet
